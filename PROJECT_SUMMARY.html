🎯 Problem Statement

Build a production-grade AI chatbot application similar to Google Gemini or ChatGPT that provides:

  • Real-time streaming AI responses (typing effect)
  • Persistent conversation history
  • Modern, responsive UI
  • Enterprise-level security and scalability

✨ Key Features

⚡ Real-Time Streaming

SSE-based streaming for ChatGPT-like typing effect. Users see AI "thinking" in real-time.

💾 Conversation History

PostgreSQL persistence with context-aware responses and export functionality.

🎤 Voice Input

Web Speech API integration for hands-free voice-to-text messaging.

📝 Notes System

Save, organize, pin, and download important AI responses.

🔒 Enterprise Security

Rate limiting, XSS prevention, SQL injection protection, security headers.

📱 PWA Support

Installable on mobile/desktop with offline capability.

🔒 Security Implementation

Rate Limiting

// 15 chat requests per minute per IP
export const chatLimiter = rateLimit({
    windowMs: 1 * 60 * 1000,
    max: 15,
    message: { error: 'Too many requests' }
});

Input Sanitization

const sanitizeInput = (input: string): string => {
    return input
        .trim()
        .slice(0, MAX_MESSAGE_LENGTH)
        .replace(/<script\b[^<]*>/gi, '');
};

Security Headers (Helmet.js)

app.use(helmet({
    contentSecurityPolicy: {
        directives: {
            defaultSrc: ["'self'"],
            scriptSrc: ["'self'"],
        },
    },
}));
